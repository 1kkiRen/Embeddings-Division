About
==================

Python script for manipulating the existing tokenizer.

The solution was tested on meta-llama/Llama-3.2-1B.

Installation
==================

.. code-block:: bash

    pip install embdiv


Requirements
==================
- Python 3.9+
- tokenizers>=0.21.0
- transformers>=4.47.0
- tqdm>=4.66.4
